GITHUB

GitHub es un sistema de gestión de proyectos y control de versiones de código, así como una plataforma de red social diseñada para desarrolladores.

En general, permite trabajar en colaboración con otras personas de todo el mundo, planificar proyectos y realizar un seguimiento del trabajo.

Existen varios aspectos que hacen que GitHub sea una opción eficiente para el control y gestión de tus proyectos:

  • GitHub permite que alojemos proyectos en repositorios de forma gratuita.
  • Los repositorios pueden ser públicos o privados.
  • Permite que puedas compartir tus proyectos de una forma mucho más fácil.
  • Te permite colaborar para mejorar los proyectos de otros y a otros mejorar o aportar a los tuyos.

GitHub tiene dos Herramientas muy útiles que principalmente hemos usado para el desarrollo de este sitio Web, estas son GitHub Desktop y GitHub Pages, con la primera podemos manejar los repositorios de manera local y posteriormente enviarlo al master y de forma inversa clonar un repositorio en nuestra computadora, y la segunda, GitHub Pages para publicar nuestro repositorio a manera de página web, pero en ello se profundiza en la siguiente lectura de esta página web, a GitHub se puede acceder a través del siguiente enlace a su web oficial: https://github.com/
