CSS3

CSS3 es la última versión que encontramos de CSS, significa: Hojas de Estilo en Cascada (Cascading Style Sheets en inglés).

Es un lenguaje declarativo que controla el aspecto de las páginas web en el navegador. El navegador aplica declaraciones de estilo CSS a los elementos seleccionados para mostrarlos correctamente. Una declaración de estilo contiene las propiedades y sus respectivos valores, los cuales determinan cómo se verá una página web.

CSS es una de las tres principales tecnologias web, junto con HTML y JavaScript, Una regla CSS es un conjunto de propiedades asociadas con un selector.

CSS es uno de los lenguajes base de la Open Web y posee una especificación estandarizada por parte del W3C. Desarrollado en niveles, CSS1 es ahora obsoleto, CSS2.1 es una recomendación y CSS3, ahora dividido en módulos más pequeños, está progresando en camino al estándar.
