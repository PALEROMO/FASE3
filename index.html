Información sobre este OVI

Este OVI fue desarrollado para poner en práctica los conocimientos acerca del desarrollo de sitios WEB en los formatos HTML5 y CSS3, por medio de un repositorio en GITHUB, que fue trabajado de manera colaborativa, por los integrantes del grupo, cuya información se encuentra en la página de autores, en el repositorio de GITHUB se trabajó por medio del aplicativo GITHUB Desktop, donde cada uno iba sincronizando sus commits al master del repositorio, para posteriormente una vez desarrollado poder realizar la publicación de la pagina en la función de GITHUB Pages, esto para poder ser visualizado lo creado, en los distintos navegadores.

Objetivos del OVI:

  • Aprender sobre los conceptos básicos de HTML5 y CSS3 y sus Estructuras.

  • Aplicar los conocimientos obtenidos sobre HTML5 y CSS3 y su semantica.

  • Desarrollar este OVI según los Mockups desarrollados en la Fase anterior.

  • Aprender el uso de los commits para los repositorios en GITHUB.

  • Aprender a trabajar de manera colaborativa en un repositorio de GITHUB y conocer sus ventajas.

  • Publicar el repositorio como sitio web en GITHUB Pages correctamente.

Temáticas a trabajar en este OVI:

En este OVI se trabajará una introducción a HTML5, CSS3, GIT, GITHUB, y GITHUB Pages por medio de unas entradas de esta página donde se encuentra contenida la información respectiva a cada temática anteriormente mencionada por medio de lecturas, en el se encontrará información básica sobre ello, también se encuentra información a manera de multimedia (audios, videos, imágenes, etc) para profundizar un poco más sobre los temas tratados, adicionalmente este sitio web cuenta con una serie de actividades que nos ayudarán a aprender y recordar lo aprendido de forma más didáctica.

Información sobre el curso Diseños de Sitios WEB

Propósitos del curso:

  • Se epera que al finalizar el curso, el estudiantes contruya sitios web, por medio de un sistema de control de versiones con lenguajes de marca en HTML5 y CSS3, para potenciar las habilidades en el desarrollo de metodologías y herramientas especializadas en linea.

Competencias generales del curso:

  • El estudiante comprende sistemas de control de versiones como GIT o GITHUB, para compartir y publicar archivos en formato HTML5 y CSS3, a través de linea de comando o por medio de la consola gráfica de GitHub Desktop.

  • El estudiante desarrolla sitios web en formato HTML5 y CSS3, para ser visualizados en diferentes navegadores teniendo en cuenta su funcionabilidad y usabilidad, por medio del uso de diferentes editores de código propuestos en el desarrollo del curso.

  • El estudiante evalua sitios web y da un plan de mejoramiento, por medio del uso de diferentes metodologías y herramientas especializadas en linea, para su respectiva validación y ajustes pertinentes.
